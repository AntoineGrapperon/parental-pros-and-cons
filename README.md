# Parental Pros & Cons

A Streamlit-based decision support tool for new parents. This application provides a structured overview of common questions, comparing different options with their respective pros, cons, and curated resources to help parents make informed choices.

## Features

- **Decision Categories:** Organized by topics like Feeding, Sleep, Gear, and Childcare.
- **Pros & Cons Comparison:** Clear, side-by-side analysis of common choices.
- **Resource Library:** Links to evidence-based articles, videos, and tools.
- **Interactive UI:** Simple navigation to find answers quickly.

## Getting Started

### Prerequisites

- Python 3.8+
- pip

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd parental-pros-and-cons
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the app:
   ```bash
   streamlit run app.py
   ```

## Tech Stack

- **Frontend/Backend:** [Streamlit](https://streamlit.io/)
- **Data Handling:** Pandas / JSON
- **Styling:** Vanilla CSS (customized within Streamlit)
